# SKU Movement Analysis Dashboard

This project analyzes SKU-level inventory movement using a Power BI dashboard designed to highlight operational patterns, velocity trends, contribution share, and inbound vs outbound balance.  
The goal is to demonstrate strong analytical capability, data modeling, and professional dashboard design for supply chain and inventory operations.

---

## 📊 Dashboard Visuals

### **1. Top Ten Inventory Movers**
This visual highlights the highest‑velocity SKUs based on total movement volume.

### **2. Contribution Share of Top Inventory Movers**

### **3. Movement Trend for Top Ten Inventory**
